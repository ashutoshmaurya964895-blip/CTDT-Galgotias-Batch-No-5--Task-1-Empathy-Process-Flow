Create a highly structured, professional Empathy Process Flowchart for a Mobile Alarm Clock App to identify critical design flaws and user pain points. 1. START (Oval): User context sleeping peacefully at night.
2. TRIGGER (Rectangle): Alarm rings at designated morning time.
3. USER BEHAVIOR (Diamond): User interacts with the screen while half-asleep, disoriented, and experiencing low cognitive awareness.
4. SYSTEM FRICTION (Rectangle): User attempts to click "Snooze" but accidentally hits "Dismiss" because both buttons feature identical sizing, proximity, and visual weight.
5. EMOTIONAL PAIN POINT (Rectangle): Alarm permanently shuts off. User falls back asleep, wakes up severely late, and feels intense panic, anxiety, and frustration.
6. INSIGHT & SOLUTION GOAL (Rectangle): Identify the design requirement for a fool-proof interface, such as a mandatory puzzle-solving step, physical shake action, or a long-swipe gesture to complete a 'Dismiss' action.
7. END (Oval): Empathy Process Flow Complete.

Use a clean modern layout with clear directional arrows connecting each step to demonstrate the user's emotional journey.
